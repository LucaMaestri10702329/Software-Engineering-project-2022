- Download the jar from <a href="https://github.com/lucalain/ing-sw-2022-Lain-Maestri-Lupo/blob/main/deliveries/jar/PSP37-1.0-SNAPSHOT.jar">here</a>

- Launch application executing:

```
$ java -jar PSP37-1.0-SNAPSHOT.jar
```
